# Dehydration Business Static Website

This is a lightweight, modern static website built with:
- TailwindCSS (CDN)
- Alpine.js (CDN)

## Quick start (local)
Just open `index.html` in your browser.

## Customize
Edit these in `index.html`:
- Brand name
- WhatsApp number (`whatsappNumber`)
- Email + address
- Optional: add images into `/assets` and replace placeholders.
